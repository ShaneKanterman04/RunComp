# RunComp 0.3.0 Plan

## Theme

Make RunComp feel alive between runs.

0.2.0 made the app feel more native and playful. 0.3.0 should focus on the engagement loop: reasons to open the app, react, compete, and keep the family race moving even on days when nobody has logged yet.

## Major Goals

### 1. Challenges

Add lightweight family challenges that create short-term goals inside the larger race.

- Weekly family mileage challenge.
- Weekend participation challenge.
- Beat last week's family total.
- Everyone logs at least one run.
- Most consistent runner of the week.
- Challenge progress card on Home.
- Challenge events in Feed.
- Challenge-complete notifications.

### 2. Better Notifications

Make push notifications feel useful, timely, and fun.

- Notify when someone logs a run.
- Notify on lead changes.
- Notify when someone is close to passing another runner.
- Notify when a family challenge is completed.
- Notify when the weekly recap is ready.
- Optional weekend nudge if nobody has logged yet.
- Avoid noisy duplicate notifications.

### 3. Runner Profiles

Turn runner cards into deeper profile views.

- Tap a runner to open a full profile.
- Show personal records:
  - Longest run
  - Fastest pace
  - Biggest week
  - Best streak
- Show achievement shelf.
- Show recent run trend.
- Show head-to-head stats against each family member.
- Show runner title and card rarity more prominently.

### 4. Weekly Recap Screen

Promote the weekly recap from a panel into a full native-feeling screen.

- Family headline.
- Total miles and runs.
- Top runner.
- Biggest run.
- Most improved.
- Best streak.
- Crowd favorite.
- Challenge winners.
- Feed-style recap moments.
- Later: shareable recap image.

### 5. Feed Moments

Make the Feed more than a raw run list.

- Runs.
- Achievements.
- Challenge progress.
- Challenge completed.
- Lead changes.
- Close-call comeback events.
- Weekly recap posted.
- Reactions on feed moments, not only runs.

### 6. Owner And Setup Polish

Make the group owner flow easier for family use.

- Invite status for each runner.
- Copy login link per runner.
- Reset runner password.
- Edit runner display name.
- Remove inactive runner.
- Clearer owner-only controls.
- Better first-run setup flow.

### 7. Backup And Export

Since this is homelab and family data, make it easy to keep a copy.

- Download JSON backup.
- Download CSV runs export.
- Basic import/restore later.
- Show last backup timestamp.
- Keep implementation simple and local-first.

### 8. Real Settings Screen

Move scattered group controls into one native-feeling Settings area.

- Race goal.
- Notification status.
- Runner management.
- Backup/export.
- Install/PWA status.
- App version.
- Logout and switch group.

## Suggested 0.3.0 Scope

Keep the release focused enough to finish cleanly:

1. Challenges.
2. Runner profile views.
3. Improved notification triggers.
4. Dedicated weekly recap screen.
5. Owner/admin polish.
6. Backup/export.

## Defer For Later

- GPS tracking.
- Background run recording.
- Route tracking.
- Public sharing.
- Complex privacy controls.
- Full social feed.

These can add a lot of complexity without improving the core family competition loop.
