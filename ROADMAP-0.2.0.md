# RunComp 0.2.0 Plan

## Theme

Make RunComp feel like a real family sports app: faster daily use, more playful comparison, better recaps, and fewer rough mobile edges.

## Core Plan

### 1. Polish Mobile PWA

- Clean up all four tabs: Home, Log, Feed, Group.
- Make each tab feel purpose-built, not like chopped-up desktop sections.
- Add better loading and sync states.
- Add small "updated just now" text after polling refreshes.
- Make delete, refresh, and settings actions less prominent than primary actions.

### 2. Upgrade Runner Cards

- Add "card backs" or a detail view when tapping a runner card.
- Show personal records:
  - Longest run
  - Fastest pace
  - Biggest week
- Add runner nicknames or titles:
  - Hill Goblin
  - Pace Menace
  - Consistency Merchant
- Add card rarity styling for milestones.

### 3. Weekly Recap 2.0

- Create a richer Sunday recap screen.
- Include:
  - Total family miles
  - Top runner
  - Biggest run
  - Most improved
  - Best streak
  - Crowd favorite
  - Fun generated headline
- Later: add a shareable recap card image.

### 4. More Fun Feed

- Improve reactions further.
- Add quick comments and taunts:
  - suspicious pace
  - respect
  - catching up
  - monster run
- Add feed event types:
  - Milestone hit
  - Lead change
  - Streak extended
  - Achievement unlocked

### 5. Comeback Meter

- Show exactly what each person needs to pass the next runner.
- Example: "Molly needs 1.8 mi to pass Shane."
- Add this to Home and runner cards.

### 6. Achievements Expansion

- Add hidden achievements.
- Add achievements for:
  - First 5K
  - Weekend run
  - Early run
  - Treadmill run
  - 3-run week
  - Comeback pass
  - New longest run
  - Family goal week

## Original Feature Ideas

- **Lead Change Alerts**: notify the group when someone takes first place.
- **Called Shot**: before a run, tap "Calling 3 mi." If the run is logged, it shows in the feed.
- **Family Rivalry of the Week**: app picks two runners to compare for the week.
- **Runner Nicknames**: automatic funny titles based on behavior.
- **Card Collection**: each runner earns new card designs as milestones unlock.
- **Sunday SportsCenter**: weekly recap written like a tiny sports headline.
- **Route Nicknames**: save common routes like "Mailbox Loop" or "Hill of Regret."
- **Streak Freeze**: one monthly save if someone misses a day.
- **Milestone Confetti Moments**: bigger celebration for PRs and goal completions.

## Suggested 0.2.0 Scope

Keep the release focused:

1. Comeback Meter
2. Weekly Recap 2.0
3. Lead-change and milestone feed events
4. Better runner-card details
5. More achievements
6. Final mobile polish pass

This would be a meaningful 0.2.0 without turning it into a huge rebuild.
